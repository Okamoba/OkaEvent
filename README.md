# OkaEvent
告知したいイベントを自由に投稿し、閲覧できるアプリです。

## 機能
- 認証。　最初にアプリを開いた時にFirebaseの認証をする。
- 投稿。Firebaseのデータベースに追加保存する。
- 編集。Firebaseのデータベースに保存されてあるデータを変更する。
- 閲覧。Firebaseのデータベースからデータを取得し表示する。
- 通知。Firebaseのデータベースが変更されたら通知を送る。

## データ構造
- event
 - イベントURL
 - イベント名前
 - 投稿者名前
 - 投稿者ID
 - 概要
 - 推しポイント
 
### ルール
- 同じURLは登録できないようにする。
- 全てNOT NULL
